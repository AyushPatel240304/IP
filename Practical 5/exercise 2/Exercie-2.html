HR TAG

Hr tag is used to display a line.



Attributes of HR tag:-

  1. Width:-
    • It is used to define the width of the line.
  2. Size:-
    • It is used to define the size of the line.
  3. color:-
    • It is used to define the color of the line.
  4. algin:-
    • It is used to define the alignement of the line.
  5. Border-radius:-
    • It is used to define the border radius of the line.
