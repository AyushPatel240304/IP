THERE ARE FIVE BACKGROUND PRORERTIES OF CSS:-

  • Background Color
  • Background Image
  • Background Repeat
  • Background Attachment
  • Background Shorthand
  1. Background Color:
    Background Color
    This tag is used to define background color of any element.
    Opacity
    This tag is used to specify the opacity/transparency of an element.It ca ntake value from 0.0 - 1.0.The lower value the more transparent.
  2. Background-image:
    Background-image:
    This tag is used to specify an image as a background for an element.
  3. Background-repeat:
    Background-repeat:
    This tag is used to specify wether an image should repeat in X-axis,Y-axis,both or none.
    Background-position:
    This tag is used to specify the position of the image.
  4. Background-attachment:
    Background-attachment:
    This tag is used to specifies whether the background image should scroll or be fixed (will not scroll with the rest of the page).
  5. Background-shorthand:
    Background-shorthand:
    This tag is used To shorten the code, it is also possible to specify all the background properties in one single property. This is called a shorthand property.
